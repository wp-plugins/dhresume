Provide a service which is like a resume directory.

== Description ==

DhResume can provide a directory service of resume coded in hResume microformat in your WordPress blog.
Just give the URL of the XML-RPC service to users, then they can publish the CV in microformat hResume directory.
The directory provided a search engine and index display.

The administration can delete, update or add a resume coded in hResume microformat.
DhResume is compatible with LinkedIn

To publish a resume in the directory, you need to have a resume in hResume microformat.
Use the plugin My-hResume and add the URL of service in the end of the form.

== Installation ==

1. Upload DhResume to your /wp-content/plugins/ directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Add the shortag [dhr_display] in a new page and publish it.
